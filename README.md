## Overview

Puppy Valentine is a fully client-side interactive experience featuring animated ASCII art, progressive interaction mechanics, persistent state, and runtime layout synchronization — all implemented without a frontend framework.

---

## Features

- Animated ASCII puppy with timed wag effect  
- Progressive “friendship” level system  
- Combo and best-combo tracking with `localStorage` persistence  
- Dynamic heart particle effects  
- Responsive layout syncing via DOM measurement  
- Accessible custom modal reset dialog  
- Fully client-side, no frameworks  

---

## Tech Stack

- HTML5  
- Modern CSS (custom properties, gradients, animation)  
- Vanilla JavaScript (ES6+)  
- `localStorage`  
- GitHub Pages deployment  

---

## Technical Highlights

- Manual state management without a framework  
- Deterministic render cycle  
- Runtime layout measurement using `getBoundingClientRect()`  
- Controlled animation timing and cleanup  
- Persistent client-side UX state
