Puppy Valentine — Interactive Web Gift
An interactive, single-page Valentine built with HTML, CSS, and vanilla JavaScript.

Features!

-Animated ASCII puppy with wag effect

-Progressive “friendship” system (level-based progression)

-Combo + best combo tracking (persisted via localStorage)

-Animated heart particle effects

-Dynamic UI layout syncing (JS-driven DOM measurement)

-Custom modal reset confirmation

-Fully client-side, no frameworks

Tech Stack!

-HTML5

-Modern CSS (custom properties, gradients, animation)

-Vanilla JavaScript

-localStorage persistence

-GitHub Pages deployment


This project demonstrates:

-DOM state management without a framework

-Animation timing control

-Layout synchronization using runtime measurements

-UI state persistence

-UX polish in a single-file architecture
